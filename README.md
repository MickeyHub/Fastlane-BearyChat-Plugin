# Fastlane-BearyChat-Plugin

A Custom lane for bearychat

## Usage
```ruby
bearychat(
  webhook: 'https://xxxxx',
  text: '',
  notification: '',
  markdown: false,
  channel: '',
  user: '',
  attachments: [
    {
      title: '',
      text: '',
      color: '',
      images:[{'url': 'https://xxxx.jpg'}]
    }
  ]
)
```
